## Criptografia e Segurança

Projeto focado na implementação prática de conceitos fundamentais de criptografia e segurança da informação, com ênfase em proteção de dados, confidencialidade e boas práticas de desenvolvimento seguro.

---

## Sobre o projeto

Este repositório reúne implementações e experimentos relacionados à segurança de sistemas, incluindo:

- Criptografia de arquivos
- Uso de algoritmos como Base64, AES e SHA256
- Derivação de chave segura (PBKDF2)
- Manipulação segura de dados sensíveis

A criptografia é uma das principais técnicas para garantir a proteção de dados contra acessos não autorizados.

---

## Objetivo

O objetivo deste projeto é consolidar conhecimentos em criptografia, assim como aplicar conceitos teóricos em código, desenvolver pensamento voltado à segurança, servir como base para estudos em segurança e IA.

---

## Funcionalidades

* Cifrar arquivos com senha
* Decifrar arquivos protegidos
* Testes de funcionamento
* Manipulação de arquivos `.txt`

---

## Tecnologias utilizadas

* Python
* Criptografia BASE64 e AES-256
* PBKDF2 (derivação de chave)
* Manipulação de arquivos

---

## Como executar "cofre_aes.py"

```bash
python cofre_aes.py cifrar texto
```

```bash
python cofre_aes.py decifrar texto.cifrado
```

```bash
python cofre_aes.py testar
```
## Em Mac: 
Somente adicionar um 3 no final de python, ex.:
```bash
python3 cofre_aes.py cifrar texto
```
```bash
python3 cofre_aes.py decifrar texto.cifrado
```

```bash
python3 cofre_aes.py testar
```
